# Kursovaya1 - Настольное приложение на JavaFX
Курсовая работа

Простое настольное приложение для визуализации объектно-ориентированной модели MUD-игры.

## Требования

- Java 11 или выше
- Maven 3.6 или выше
- JavaFX 17 (зависимости загружаются автоматически через Maven)

## Запуск приложения

### Способ 1: Через Maven (рекомендуется)

```bash
mvn clean javafx:run
```

### Способ 2: Через IDE (Giga IDE / IntelliJ IDEA)

Если при запуске возникает ошибка "JavaFX runtime components are missing", выполните следующие шаги:

1. **Настройка VM options в конфигурации запуска:**
   - Откройте конфигурацию запуска (Run Configuration)
   - В поле "VM options" добавьте:
   ```
   --module-path "путь_к_javafx_lib" --add-modules javafx.controls,javafx.fxml
   ```
   
   Или используйте автоматическое определение пути через Maven:
   ```
   --module-path "${project.basedir}/target/classes" --add-modules javafx.controls,javafx.fxml
   ```

2. **Альтернативный способ - через переменные окружения:**
   - Установите переменную окружения `JAVAFX_HOME` на путь к JavaFX SDK
   - Или используйте Maven для управления зависимостями

3. **Самый простой способ:**
   - Используйте Maven для запуска: `mvn javafx:run`
   - Или создайте конфигурацию запуска Maven с целью `javafx:run`

## Структура проекта

```
src/main/java/com/mudgame/
├── model/
│   ├── TGameObject.java    # Базовый абстрактный класс
│   ├── TRoom.java          # Класс локации
│   ├── TPlayer.java        # Класс игрока
│   ├── TItem.java          # Базовый класс предмета
│   ├── TWeapon.java        # Класс оружия
│   └── TNPC.java           # Класс NPC
└── MUDGame.java            # Главный класс с JavaFX интерфейсом
```

## Использование

1. Запустите приложение
2. Нажмите кнопку "Войти в локацию"
3. Используйте кнопки для управления:
   - **Атаковать** - атакует NPC в комнате
   - **Переместиться** - случайно перемещает игрока и NPC
   - **Поднять предмет** - поднимает предмет из комнаты в инвентарь

## Примечания

- Все взаимодействия реализованы через методы объектов
- Код содержит подробные комментарии на русском языке
- Архитектура спроектирована для создания CRC-карточек и диаграмм классов
